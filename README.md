# Restaurant Analytics Dashboard & Data Analysis Report

This project delivers an interactive Power BI dashboard analyzing a multi-country Zomato restaurant dataset to provide insights for customers, analysts, and restaurant owners.

## Objectives

- **User-Centric Exploration:**  
  Enables filtering of restaurants by country, city, cuisine, price range, booking options, and delivery availability. Provides search capabilities to find restaurants including details like name, city, price range, and rating.

- **Performance Tracking:**  
  Highlights key metrics such as average cost, number of restaurants, average rating, maximum rating, and total votes across cities and countries.

- **Trend Analysis:**  
  Visualizes changes over time in restaurant counts and ratings by year, helping understand rating improvements or declines.

- **Comparative Insights:**  
  Allows comparison of restaurant availability and performance across cities (e.g., New Delhi, Gurgaon, Noida) and cuisines.

## Dashboard Overview

- **Filters and Slicers:**  
  Price ranges (14 levels), table booking options (Yes/No), online booking (Yes/No), delivery availability (Yes/No), country, city, currency, and cuisine filters.
  
- **Restaurant Details Table:**  
  Displays restaurant name, city, price range, and rating for direct exploration.

- **Key Metrics:**  
  KPI cards and gauges for average cost, maximum rating, average rating, total votes, and total restaurants.

- **Trend Visualization:**  
  Line charts showing total restaurants and average rating trends over years (2010 to 2017).

- **City Comparison:**  
  Bar charts presenting restaurant counts across major cities.

## Value to Users

- For **Customers:** Simplifies finding restaurants that match personal preferences and requirements.

- For **Analysts:** Provides insights into industry growth, preferences, and performance.

- For **Restaurant Owners:** Benchmarks restaurants against competitors on cost, rating, and popularity.

## Technologies Used

- Power BI for interactive data visualization and dashboard creation.

## Installation and Usage

1. Open the Power BI Desktop application.
2. Load the restaurant dataset (Zomato data).
3. Use provided slicers and filters to explore the restaurant landscape according to your criteria.
4. Analyze the visualized trends, KPIs, and comparisons for insights.
